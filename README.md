# EXPENSIFY — Full-Stack Budget Tracker
## FEATURES

- **User Accounts** — JWT-authenticated login & signup with slide animation
- **Configurable Split %** — Slider to set how surpluses/deficits/external income split between Savings and Personal
- **Manual Weekly Budget Allocation** — Set exact amounts per week (Fixed+Wk1, Wk2, Wk3, Wk4)
- **Three Dashboards** — Input, Needs (weekly tracking), Income (flow + charts)
- **End Week Logic** — Close weeks; surplus/deficit handled per your split setting
- **External Income** — Add and split between Savings/Personal at your chosen ratio
- **Charts** — Bar chart (weekly needs) and Pie chart (money flow)
